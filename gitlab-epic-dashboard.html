How to Use EpicDash

1. Get your GitLab Personal Access Token with api scope.
2. Paste a Group URL (recommended) or an Epic URL.
3. Click Load Dashboard.
4. Use filters on the left and explore the tree view.
Token is kept only in memory and cleared when you edit input.
